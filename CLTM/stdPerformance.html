Student Learning Behaviour for MOOC Data from a Psychology Course on Coursera

This page shows the behaviour of the hidden groups of students for different concept (knowledge component) clusters. There are 10 concept clusters in the knowledge component latent tree. Each of the below bottons correspond to a concept cluster.

If you click on one of the concept clusters you can see the inferred state of groups of hidden variables for that cluster. The hidden variables belonging to the same cluster are shown by the same color in the student latent tree. The average inferred state of the hidden variables is shown in the graphs below the tree for each concept cluster and student cluster.
